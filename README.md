## Playtime Adventure

A game made for [Brackeys Game Jam 2022.1](https://itch.io/jam/brackeys-7/rate/1417955)
Created with collaboration from [NekotoArts](https://nekotoarts.github.io/)

First ever 3d game. It's a bit rough around the edges, but understood how adding a 3rd dimention adds a LOT more things to learn.
